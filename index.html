Temas teóricos adicionales que serán evaluados:

  • ¿Qué es un navegador

    Un navegador web es un programa o aplicación que permite acceder, interpretar y visualizar páginas web en Internet. Ejemplos: Google Chrome, Mozilla Firefox y Microsoft Edge.

  • ¿Qué es un metadato?

    Un metadato es información que describe las características o propiedades de un archivo, documento o conjunto de datos. Su función es ayudar a identificar, organizar y comprender mejor la información.

    Por ejemplo, en cualquier archivo de computadora los metadatos pueden ser:

    • Fecha de creación.
    • Fecha de modificación.
    • Tamaño del archivo.
    • Ruta o ubicación donde se encuentra guardado.

    En los documentos HTML también existen metadatos. En este caso, los metadatos se alojan internamente dentro del mismo documento HTML mediante etiquetas <meta> ubicadas en la sección <head>.

  • ¿Cómo interpreta HTML un navegador?

    Un navegador posee motores de interpretación integrados que analizan códigos como HTML, CSS y JavaScript para representar páginas web y ejecutar funcionalidades interactivas.

  • ¿Qué es la importancia semántica?

    La importancia semántica en HTML se refiere al uso correcto de las etiquetas según el significado del contenido que representan. Esto permite que los navegadores y, especialmente, los motores de búsqueda interpreten mejor la estructura y el contenido de una página web.

    La semántica es muy importante para los motores de búsqueda como Google, ya que les facilita analizar e indexar correctamente la información de una página. Gracias a esto, los buscadores pueden comprender mejor qué contenido es principal, qué partes son navegación, artículos, títulos o información secundaria, mejorando así el posicionamiento SEO en los resultados de búsqueda.
